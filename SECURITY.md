# Security Policy

## Supported Versions

This project is a research prototype developed for the Google AI Agents Intensive.
Only the current version (main branch) is actively supported and maintained.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in MindShield AI, please do NOT open a public issue.
Instead, report it immediately via email to: [er6nsona19@gmail.com]

I will acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible.
